Welcome to my Final Project Page!!

Hello!! Sophia and myself decided to make an educational app to teach children how to draw shapes. We thought an educational
app wpuld demonstrate what we learned the best!! In order to work the app you have to press the start button. After that you
select which shape you would like to learn how to draw. Once you do that you choose which color you would like to draw with.
Then learn how to draw!! If you would like to choose a different color to draw in, there are three options for you!! Have fun!!
